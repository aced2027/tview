## ⚡ QUICK WORKFLOWS

### Workflow 1: Support & Resistance
```
1. Select Horizontal Line
2. Click at support level
3. Change color to Green
4. Click at resistance level
5. Change color to Red
6. Lock both lines
✓ Now you have clear levels!
```

### Workflow 2: Fibonacci Setup
```
1. Select Fibonacci Retracement
2. Click at swing low (bottom)
3. Click at swing high (top)
4. Shows: 0%, 23.6%, 38.2%, 50%, 61.8%, 100%
5. Watch for bounces at 38.2% - 61.8%
✓ Find pullback levels!
```

### Workflow 3: Trade Entry
```
1. Rectangle → Mark entry zone
2. Horizontal → Mark exit target
3. Text → Add notes
4. Callout → Entry signal
5. Lock all → Protect analysis
✓ Trade plan complete!
```

### Workflow 4: Multi-Timeframe
```
1. Draw levels on 1H chart
2. Lock drawings
3. Switch to 4H chart
4. Add levels with different color
5. Find confluence zones
✓ Identify key levels!
```
